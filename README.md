# Muzita Legacy - Elderly Care PVO Website

A modern, responsive website for Muzita Legacy, a Private Voluntary Organisation supporting Zimbabwe's elderly with home visits, health support, and social connections.

## 🌟 Features

- **Purple-themed design** with accessibility focus for elderly users
- **6 Complete Pages**: Home, About, Services, Conditions, Gallery, Contact
- **Responsive Design**: Works on all devices
- **Interactive Elements**: Testimonials slider, accordions, lightbox gallery
- **Contact Forms**: Volunteer, donation, and partnership inquiries
- **SEO Optimized**: Meta tags and structured content

## 🚀 Deployment on Railway

### Prerequisites
- Railway account
- GitHub account
- This project code

### Steps to Deploy

1. **Push to GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit - Muzita Legacy website"
   git branch -M main
   git remote add origin https://github.com/yourusername/muzita-legacy.git
   git push -u origin main
   ```

2. **Deploy on Railway**
   - Go to [railway.app](https://railway.app)
   - Click "New Project" → "Deploy from GitHub repo"
   - Select your repository
   - Railway will automatically detect the static site
   - Click "Deploy"

3. **Configure Settings**
   - Set environment variables if needed
   - Configure custom domain (optional)
   - Enable auto-deploys from main branch
